📊 Sobre o projeto

Aplicação web desenvolvida com HTML, CSS e JavaScript para gerenciamento de músicas.
O sistema permite cadastrar músicas informando nome, álbum, duração e data de lançamento, exibindo os dados em uma tabela dinâmica.Também é possível selecionar músicas para edição, facilitando a atualização das informações. 
Projeto desenvolvido no contexto da disciplina de Programação Web na FATEC, com foco na aplicação prática de conceitos fundamentais como manipulação do DOM, eventos em JavaScript e organização de dados. 

🚀 Tecnologias utilizadas

HTML
CSS
JavaScript
jQuery
